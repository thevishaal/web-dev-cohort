Master JavaScript built-in methods and core concepts through hands-on **kahaniya & coding labs**.

# 📋 Prerequisites

Before starting, make sure you have installed:

- [Node.js](https://nodejs.org/) (v20 or higher)
- [Git](https://git-scm.com/)
- A code editor like VS Code

---

# ⚡ Getting Started

## 1. Accept the Assignment

Click the GitHub Classroom invitation link shared by your instructor.

This will create your **personal copy** of the assignment repository under your GitHub account.

---

## 2. Clone Your Repository

```bash
git clone <your-repo-url>
cd <your-repo-folder>
```

Replace `<your-repo-url>` with your GitHub Classroom repository URL.

You can find it from the green **Code** button on GitHub.

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Run Tests

```bash
npm test
```

Initially, all tests will fail ✅  
That's expected.

Your goal is to write code and make all tests pass.

---
