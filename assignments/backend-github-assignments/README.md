# Backend GitHub Assignments 🚀

Build real-world backend skills through hands-on labs focused on APIs, authentication, CRUD, and file uploads.

---

# 📋 Prerequisites

Before starting, make sure you have installed:

- [Node.js](https://nodejs.org/) (v20 or higher)
- [Git](https://git-scm.com/)
- [Docker](https://www.docker.com/) (if required by a lab)
- A code editor like VS Code

---

# ⚡ Getting Started

## 1. Accept the Assignment

Click the GitHub Classroom invitation link shared by your instructor.

This will create your personal copy of the backend assignment repository under your GitHub account.

---

## 2. Clone Your Repository

```bash
git clone <your-repo-url>
cd <your-repo-folder>
```

Replace `<your-repo-url>` with your GitHub Classroom repository URL.

You can find it from the green Code button on GitHub.

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Run the Project

Each backend lab may use a different command depending on the assignment.

Common commands include:

```bash
npm test
```

or

```bash
npm run dev
```

Read the lab-specific README carefully before starting.

---

# 📚 Important Instructions

⚠️ Before starting any lab:

👉 Carefully read the README file inside every lab folder.

Each lab contains:

- Problem statements
- API requirements
- Examples
- Rules & constraints
- Expected behavior

Reading the README properly will save you a lot of debugging time.
